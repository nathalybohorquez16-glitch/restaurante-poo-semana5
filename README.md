## Descripción

Este proyecto consiste en un sistema básico de gestión de restaurante desarrollado en Python utilizando Programación Orientada a Objetos.

El sistema permite registrar productos y clientes mediante clases independientes y una clase de servicio que administra la información.

## Estructura del proyecto

```
restaurante-poo-semana5/
│
├── modelos/
│   ├── __init__.py
│   ├── cliente.py
│   └── producto.py
│
├── servicios/
│   ├── __init__.py
│   └── restaurante.py
│
├── main.py
└── README.md
```
